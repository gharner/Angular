All elements with the title attribute containing the word "rose" have a red font.

Show in Stackblitz 
[title~='rose'] {{ '{' }}
     color: red !important
{{ '}' }}
A rose by any other name would smell as sweet

> is a CSS child selector

Show in Stackblitz 
p>span {{ '{' }}
     color: blue !important;
{{ '}' }}

Having one child makes you a parent; having two you are a referee.


Double-colon selectors are pseudo-elements.

     text-transform: uppercase;
{{ '}' }}

Single-colon selectors are pseudo-class.
